# 🎨 Crystal FailSafe - IT Support Ticketing System
Live-Demo https://crystal-failsafe.vercel.app/ 

AI-powered IT ticketing system for Crystal Bridges Museum. Automatically processes employee emails, classifies tickets with AI, and provides IT staff with a modern dashboard.

![React](https://img.shields.io/badge/React-18.3-blue) ![Django](https://img.shields.io/badge/Django-4.2-green) ![Status](https://img.shields.io/badge/Frontend-Complete-success) ![Backend](https://img.shields.io/badge/Backend-In%20Progress-yellow)

## 🎯 Overview

**Problem:** IT departments waste 10+ hours/week manually sorting support emails.

**Solution:** Crystal FailSafe automatically monitors emails, uses AI to classify tickets, and gives IT staff a beautiful dashboard to manage everything.

## ✨ Features

- 🔐 **Authentication** - Login/signup with approval workflow
- 📊 **Dashboard** - Real-time ticket stats with filtering
- 🎫 **3-Tab Ticket Modal**
  - Update status, priority, assign to staff
  - View original email with attachments
  - Reply directly to employees
- 📈 **Analytics** - Performance metrics and trend charts
- 🤖 **AI Chatbot** - Smart ticket search
- 🎨 **Beautiful UI** - Crystal Bridges branded, fully responsive

## 🛠️ Tech Stack

**Frontend (✅ Complete):**
- React 18.3, Vite, Tailwind CSS, React Router

**Backend (🚧 In Progress):**
- Django 4.2, PostgreSQL, Claude AI, Email monitoring

## 🚀 Quick Start
```bash
# Clone repo
git clone https://github.com/YOUR-USERNAME/crystal-failsafe.git
cd crystal-failsafe/frontend

# Install & run
npm install
npm run dev

# Open http://localhost:5173
# Login with any email/password (mock auth)
```
